What is Run 3?

Run 3 is the third installment in the legendary Run series, developed by Joseph Cloutier and published on Coolmath Games. It's one of the most-played browser games in school history, beloved for its simple controls, deep gameplay, and the unique mechanic of rotating tunnels in space.

Unlike most endless runners, Run 3 has two distinct modes. Explore Mode gives you a map of the alien tunnel system with specific levels to complete — each one introducing new challenges like crumbling tiles, moving platforms, and branching paths. Infinite Mode is the classic endless survival run where you go as far as possible without falling into the void.

How to Play Run 3

  • Move left/right using ← → arrow keys or A / D.
  • Jump with Space or the ↑ up arrow key.
  • When you run off the side of the tunnel, the whole tunnel rotates — use this to reach platforms on the other side.
  • Avoid the dark gaps in the tunnel floor — falling through them ends your run.
  • Collect power cells (glowing orbs) to unlock new characters.
  • In Explore Mode, reach the end of each level to progress on the map.

Tips for Getting Further in Run 3

1. Use tunnel rotation intentionally

Don't just run straight — deliberately run off the side of the tunnel to rotate it. This is the key mechanic that separates beginners from advanced players. A well-timed rotation can bridge a gap that looks impossible.

2. Look for the safest path, not the fastest

In Explore Mode especially, slow down and plan your route. It's better to take a longer path around a gap than to rush and fall through.

3. Collect power cells consistently

Don't ignore the glowing power cells. They're the only currency for unlocking new characters, and some characters (like the Skater) make certain levels dramatically easier.
